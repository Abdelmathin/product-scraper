# product-scraper

# Python 2.7:

```bash
curl https://bootstrap.pypa.io/pip/2.7/get-pip.py -o get-pip.py
```

```bash
python -m pip install --upgrade pip
```

```bash
python -m pip install requests
```

```bash
python main.py
```

# Python 3:

```bash
pip3 install --upgrade pip
```

```bash
pip3 install --upgrade urllib3
```

```bash
pip3 install requests
```

```bash
python3 -u main.py
```
